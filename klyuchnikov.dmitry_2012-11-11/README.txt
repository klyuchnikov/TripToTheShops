1. Оптимизация по цене - дело простое, выборка продуктов с минимальной ценой. Оптимизация по дистанции у меня построена следующим образом - сначала производится поиск всех комбинаций покупки нужного списка товаров в магазинах, а потом по имеющимся графам магазинов выбирается граф с наименьшим путем (с минимизацией обхода в самом графе(эйлеровы циклы)).
2. Особенных указаний не имеет.
3. Для запуска программы требуется .NET Framework 4 Client Profile.
